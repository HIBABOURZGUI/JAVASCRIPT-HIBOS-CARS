Guide d'Installation du Logo Hibos Car

Hibos Car

Votre nouveau logo sera affiché ici

1

Préparez votre logo

  • Format: PNG (recommandé) ou JPG
  • Taille: 300x300px minimum (carré idéal)
  • Fond transparent si possible
  • Couleurs: Or (#B8860B) et blanc
  • Nom du fichier: hibos_car_logo.png
2

Emplacement du fichier

Placez votre logo dans le dossier suivant:

assets/images/hibos_car_logo.png

Ce chemin est relatif à la racine du projet Hibos Car

3

Fichiers à mettre à jour

Le logo sera automatiquement utilisé dans ces pages:

  • index.html - Page de connexion
  • register.html - Page d'inscription
  • catalogue.html - Catalogue
  • dashboard.html - Tableau de bord admin
  • category-selection.html - Sélection de catégorie
  • rental-history.html - Historique
  • payment-system.html - Paiements
  • Et toutes les autres pages...
4

Vérification

Après avoir ajouté votre logo:

  1. Videz le cache de votre navigateur (Ctrl+F5)
  2. Actualisez les pages de l'application
  3. Vérifiez que le logo s'affiche correctement
  4. Testez les effets hover sur le logo
Astuce

Le logo actuel utilise logo-location2.png. Vous pouvez remplacer ce fichier directement ou utiliser le nouveau nom hibos_car_logo.png pour une meilleure organisation.

Important

Assurez-vous que le nom du fichier correspond exactement à hibos_car_logo.png (minuscules, underscores, pas d'espaces) pour garantir que toutes les références fonctionnent correctement.
