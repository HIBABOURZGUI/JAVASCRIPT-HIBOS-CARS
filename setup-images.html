Configuration des Images - Hibos Car

Instructions pour ajouter vos images
  1. Logo: Placez votre logo Hibos Car dans assets/images/ avec le nom hibos_car_logo.png
  2. Photos de voitures: Placez les photos des voitures dans assets/car_images/ avec les noms spécifiés ci-dessous
  3. Les formats recommandés: JPG, PNG (max 2MB par image)
  4. Dimensions recommandées: 800x600px pour les voitures, 300x300px pour le logo

Étapes Suivantes

1. Placez vos images
  • Copiez votre logo dans assets/images/hibos_car_logo.png
  • Copiez les photos de voitures dans assets/car_images/
  • Assurez-vous que les noms correspondent exactement
2. Mettez à jour les références
  • Le système utilisera automatiquement vos nouvelles images
  • Actualisez la page pour voir les changements
  • Les images manquantes montreront des icônes par défaut
Astuce: Une fois les images ajoutées, videz le cache de votre navigateur (Ctrl+F5) pour voir les changements immédiatement.
